# Snake

A classic Snake game built with Godot 4.5.

## Gameplay

- Control a snake that grows longer each time it eats food
- Avoid hitting the walls or your own tail
- Score points and level up as you eat more food
- The game speeds up with each level

## Controls

| Action | Keys |
|--------|------|
| Move Up | W / Arrow Up |
| Move Down | S / Arrow Down |
| Move Left | A / Arrow Left |
| Move Right | D / Arrow Right |

## Features

- Grid-based movement on a 30x20 cell playfield
- Score tracking with 10 points per food
- Level progression every 50 points
- Increasing speed with each level
- Game over and level up screens

## Requirements

- Godot 4.5

## Running the Game

1. Open the project in Godot 4.5
2. Press F5 or click the Play button
